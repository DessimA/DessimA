## Projetos

### Conversor de Moedas
Consultar cotações de moedas e criptomoedas deveria ser simples e rápido. Esse projeto faz exatamente isso via terminal, sem anúncios e sem ruído, com histórico das conversões feitas na sessão para quem precisa de referência rápida.

[Ver repositório](https://github.com/dessima/Conversor-de-Moedas)

### QuizLab : Plataforma de Simulados
A maioria das plataformas de simulado exige cadastro, envia seus dados para servidores e te distrai com interface pesada. O QuizLab resolve isso: crie ou importe seus simulados, estude com feedback imediato ou em modo prova cronometrado, e tudo fica salvo no seu próprio navegador. Nenhum dado sai da sua máquina.

[Acessar](https://quizlab-chi.vercel.app/) | [Ver repositório](https://github.com/dessima/quizlab)

---

### Galeria NASA
Torna acessível o acervo visual da NASA : imagens do dia e fotos dos rovers em Marte : em uma interface organizada e com sistema de favoritos. Um projeto de frontend que exercita consumo de APIs reais com dados que mudam diariamente.

[Acessar](https://dessima.github.io/NASA_Galery/) | [Ver repositório](https://github.com/dessima/NASA_Galery)

### Smart Class Q&A
Em aulas ao vivo com muitos alunos, dúvidas se perdem no chat. Esse projeto resolve isso: alunos enviam perguntas por uma interface simples, e um sistema com IA (Amazon Comprehend) filtra automaticamente o que é uma dúvida real do que é só comentário social : entregando para o professor apenas o que precisa de atenção, com alerta sonoro em tempo real. Desenvolvido como projeto final do AWS re/Start.

[Ver repositório](https://github.com/dessima/smartclass-qa)

---

### FórumHub
Comunidades técnicas perdem conhecimento quando não há moderação e histórico confiável. O FórumHub é uma plataforma completa de fórum onde tópicos podem ser marcados como resolvidos, conteúdo excluído fica auditado e pode ser restaurado, e moderadores conseguem agir sem destruir o histórico da comunidade. O projeto foi feito de ponta a ponta : backend, frontend e infraestrutura.

[Ver repositório](https://github.com/DessimA/forum-hub)

---

### GENEFIC Cloudworks
Repositórios no GitHub são provas de habilidade, mas sistemas de triagem automática (ATS) não conseguem lê-los. O GENEFIC resolve essa assimetria: conecta-se ao GitHub, audita o código de um projeto e gera uma narrativa profissional baseada em evidências reais : padrões de arquitetura, qualidade de testes, complexidade de infraestrutura : traduzindo tudo para uma linguagem que recrutadores e gestores entendem. Projeto construído com arquitetura Serverless na AWS.

[Ver repositório](https://github.com/DessimA/genefic-cloudworks)
